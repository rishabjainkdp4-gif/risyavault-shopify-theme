# Risyavault - Premium Wellness Shopify Theme

A luxurious, modern, and conversion-focused Shopify theme designed for premium health, wellness, and self-care products.

## ✨ Features

✅ **Modern Design**
- Clean, minimalist aesthetic
- Spacious layouts with soft shadows
- Rounded cards and smooth animations
- Premium feel inspired by Apple and luxury brands

✅ **Conversion Optimized**
- Sticky add-to-cart on mobile
- Frequently bought together section
- Customer reviews and ratings
- Trust-building elements throughout

✅ **User Experience**
- Advanced product filtering
- Search bar with autocomplete
- Wishlist functionality
- Recently viewed products
- Mobile-first responsive design

## 🎨 Brand Colors

- Primary: White (#FFFFFF)
- Secondary: Off-white (#F8F8F8)
- Text: Dark Charcoal (#2C2C2C)
- Accent 1: Soft Green (#A8D5BA)
- Accent 2: Premium Blue (#4A90E2)
- Accent 3: Muted Teal (#5B9FA8)

## 🚀 Quick Start

1. Download this theme
2. Go to Shopify Admin → Online Store → Themes
3. Click "Add theme" → "Upload code"
4. Upload the theme folder
5. Customize and publish!
